# Intermediate Exercises

Use this directory once you are comfortable writing short Python programs on
your own.

## Contents

* applied notebooks such as `fits.ipynb`, `gaussian_distribution.ipynb`,
  `unique_distribution.ipynb`, and `protein_hydorphobicity.ipynb`;
* `practice`, which contains longer template exercises and the data files they
  need.

## Suggested order

* `gaussian_distribution.ipynb`
* `unique_distribution.ipynb`
* `fits.ipynb`
* `practice/tmpl_factorial.py`
* `practice/tmpl_repetitions.py`
* `practice/tmpl_find_longest.py`
* `practice/tmpl_create_curves.py`
