# Beginner Exercises

Start here after completing the main teaching notebooks.

## Contents

* `py4bio_2.ipynb` to `py4bio_7.ipynb`: chapter-based guided exercise sets.
* `practice`: short extra practice notebooks and template scripts.
* `UseModules`: a small example of splitting code across modules.
* `PythonForBiologists`: supporting chapter material and example files.
