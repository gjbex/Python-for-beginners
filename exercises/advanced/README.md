# Advanced Exercises

These exercises are good stretch material once the basics of Python feel
routine.

## Contents

* algorithmic notebooks such as `bin_search.ipynb`, `quicksort.ipynb`,
  `n_queens.ipynb`, `de_bruijn_graph.ipynb`, and `smith_waterman.ipynb`;
* `practice`, which contains more exploratory notebook exercises;
* `AdvancedPythonForBiologists`, which contains additional advanced examples
  and exercises.

## When to use this directory

Use these notebooks after you are comfortable with:

* loops and conditionals;
* functions;
* lists and dictionaries;
* reading and writing files;
* debugging small programs without step-by-step guidance.
